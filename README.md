# BlockBets
Um único executável que bloqueia localmente todos os sites de apostas que são regularizados no Brasil, adicionando eles ao arquivo hosts do Windows

## Funcionalidades
- Bloqueia sites de apostas regulares no Brasil
- Utiliza o arquivo hosts do Windows para bloquear sites
- Executável único para facilitar a utilização

## Como Usar
 * Baixe o arquivo executável
 * Adicione a pasta do executável as restrições do seu anti-vírus
 * Execute o arquivo executável
 * Aguarde alguns segundos para que os sites sejam bloqueados
 * Verifique se os sites estão bloqueados acessando-os no navegador
 * Caso não funcione, tente reiniciar o computador ou limpar o cache do navegador

# Como criar seu executável
Para criar seu próprio executável, siga os seguintes passos:
1. Clone o repositório do projeto
2. Execute o arquivo build_script.bat
3. Abra o arquivo BlockBets.exe na pasta dist na raiz do projeto
