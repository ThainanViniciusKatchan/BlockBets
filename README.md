# BlockBets
Um único executável que bloqueia localmente todos os sites de apostas que são regularizados no Brasil, 
adicionando eles ao arquivo hosts do Sistema

## Funcionalidades
- Bloqueia sites de apostas regulados no Brasil
- Utiliza o arquivo hosts do Windows para bloquear sites
- Executável único para facilitar a utilização
- Atualiza automaticamente a lista atualizada de sites de apostas
- Suporte para Windows e Linux

## Como Usar Windows
 * Baixe o arquivo executável
 * Adicione a pasta do executável as restrições do seu anti-vírus
 * Execute o arquivo executável
 * Aguarde alguns segundos para que os sites sejam bloqueados
 * Verifique se os sites estão bloqueados acessando-os no navegador
 * Caso não funcione, tente reiniciar o computador ou limpar o cache do navegador

## Como Usar Linux
 * baixe o arquivo BlockBetsLinux.zip
 * Extraia o mesmo em uma pasta
 * Execute o StartBlockBet.sh
 * Digite a sua Senha Root
 * Deixe o script trabalhar 
 * Verifique se os sites estão bloqueados acessando-os no navegador
 * Verifique se os sites estão bloqueados acessando-os no navegador


# Como criar seu executável
Para criar seu próprio executável, siga os seguintes passos:
1. Clone o repositório do projeto
2. Execute o arquivo build_script.bat no Windows ou o arquivo Build_Linux.sh no Linux
3. Abra o arquivo BlockBets.exe na pasta dist/win/ na raiz do projeto ou no linux o arquivo StartBlockBet.sh na pasta dist/Linux
